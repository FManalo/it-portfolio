üîß How to Update Your Projects

Step 1: Create GitHub Repositories

If you don't have these repositories yet, create them on GitHub:

+ Create New Repository

Tip: Use descriptive names that match your project titles for better organization.

Step 2: Update GitHub Links in Portfolio

Edit the file index.php and replace the GitHub URLs with your actual repository links.

Look for lines like:

<a href="https://github.com/FrancesManalo/it-portfolio" target="_blank" class="photo-btn secondary">GitHub Repo</a>

Replace FrancesManalo with your actual GitHub username and update repository names.

Step 3: Customize Project Information

You can also update:

  • Project Titles: Change the <h3> tags to match your actual project names
  • Descriptions: Update the <p> tags with your project descriptions
  • Technologies: Modify the <span class="tag"> elements to show the actual technologies used
  • Live Demo Links: Update the "View Live" buttons if you have deployed versions

Step 4: Add Project Screenshots

Replace the placeholder images with screenshots of your actual projects:

  • Take screenshots of your projects
  • Save them in the assets/images/ folder
  • Update the src attributes in the <img> tags
